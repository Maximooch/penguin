   Executive Summary



   Implement an interactive file picker modal in the Penguin CLI to enable users to browse and select files/directories for attachment to messages. This feature addresses the gap between manual path typing ( @src/main.py ) and
   drag-and-drop, providing a discoverable, keyboard-driven file selection experience.


     • *Key Benefits:**
     • Discoverability: Users can explore project structure without memorizing paths
     • Accuracy: Eliminates typos in file paths
     • Efficiency: Multi-select support for batch operations
     • Consistency: Matches patterns from Gemini CLI and modern TUIs


     • --

   Selection Modes


     • *Single Select (Default)**
     • Navigate with arrow keys
     • Press Enter to confirm
     • Selected file appears in input as  @src/index.tsx 


     • *Multi Select**
     • Space to toggle selection
     • Enter to confirm all
     • Selected files appear as  @src/index.tsx @src/types.ts 


     • *Directory Select**
     • Right arrow to expand directory
     • Left arrow to collapse/go up
     • Selecting a directory attaches all files within (with confirmation)

   Step 2.1: Create useFilePicker Hook
     • *File:**  src/ui/hooks/useFilePicker.ts 



   ╭──────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────╮
   │  import { useState, useCallback } from 'react';                                                                                                                                                                                          │
   │  import type { FileEntry } from '../components/FilePicker/types';                                                                                                                                                                        │
   │  import fs from 'fs/promises';                                                                                                                                                                                                           │
   │  import path from 'path';                                                                                                                                                                                                                │
   │                                                                                                                                                                                                                                          │
   │  export function useFilePicker() {                                                                                                                                                                                                       │
   │    const [isOpen, setIsOpen] = useState(false);                                                                                                                                                                                          │
   │    const [currentPath, setCurrentPath] = useState(process.cwd());                                                                                                                                                                        │
   │    const [files, setFiles] = useState<FileEntry[]>([]);                                                                                                                                                                                  │
   │    const [selectedFiles, setSelectedFiles] = useState<Set<string>>(new Set());                                                                                                                                                           │
   │    const [expandedDirs, setExpandedDirs] = useState<Set<string>>(new Set());                                                                                                                                                             │
   │    const [currentIndex, setCurrentIndex] = useState(0);                                                                                                                                                                                  │
   │                                                                                                                                                                                                                                          │
   │    const loadDirectory = useCallback(async (dirPath: string) => {                                                                                                                                                                        │
   │      try {                                                                                                                                                                                                                               │
   │        const entries = await fs.readdir(dirPath, { withFileTypes: true });                                                                                                                                                               │
   │        const fileEntries: FileEntry[] = await Promise.all(                                                                                                                                                                               │
   │          entries                                                                                                                                                                                                                         │
   │            .filter(entry => !entry.name.startsWith('.')) // Hide hidden files                                                                                                                                                            │
   │            .map(async (entry) => {                                                                                                                                                                                                       │
   │              const fullPath = path.join(dirPath, entry.name);                                                                                                                                                                            │
   │              const stats = await fs.stat(fullPath);                                                                                                                                                                                      │
   │              return {                                                                                                                                                                                                                    │
   │                name: entry.name,                                                                                                                                                                                                         │
   │                path: fullPath,                                                                                                                                                                                                           │
   │                type: entry.isDirectory() ? 'directory' : 'file',                                                                                                                                                                         │
   │                size: stats.size,                                                                                                                                                                                                         │
   │                modified: stats.mtime,                                                                                                                                                                                                    │
   │              };                                                                                                                                                                                                                          │
   │            })                                                                                                                                                                                                                            │
   │        );                                                                                                                                                                                                                                │
   │                                                                                                                                                                                                                                          │
   │        // Sort: directories first, then files                                                                                                                                                                                            │
   │        fileEntries.sort((a, b) => {                                                                                                                                                                                                      │
   │          if (a.type !== b.type) {                                                                                                                                                                                                        │
   │            return a.type === 'directory' ? -1 : 1;                                                                                                                                                                                       │
   │          }                                                                                                                                                                                                                               │
   │          return a.name.localeCompare(b.name);                                                                                                                                                                                            │
   │        });                                                                                                                                                                                                                               │
   │                                                                                                                                                                                                                                          │
   │        setFiles(fileEntries);                                                                                                                                                                                                            │
   │        setCurrentIndex(0);                                                                                                                                                                                                               │
   │      } catch (error) {                                                                                                                                                                                                                   │
   │        console.error('Failed to load directory:', error);                                                                                                                                                                                │
   │        setFiles([]);                                                                                                                                                                                                                     │
   │      }                                                                                                                                                                                                                                   │
   │    }, []);                                                                                                                                                                                                                               │
   │                                                                                                                                                                                                                                          │
   │    const open = useCallback((initialPath?: string) => {                                                                                                                                                                                  │
   │      const startPath = initialPath || process.cwd();                                                                                                                                                                                     │
   │      setCurrentPath(startPath);                                                                                                                                                                                                          │
   │      setIsOpen(true);                                                                                                                                                                                                                    │
   │      loadDirectory(startPath);                                                                                                                                                                                                           │
   │    }, [loadDirectory]);                                                                                                                                                                                                                  │
   │                                                                                                                                                                                                                                          │
   │    const close = useCallback(() => {                                                                                                                                                                                                     │
   │      setIsOpen(false);                                                                                                                                                                                                                   │
   │      setSelectedFiles(new Set());                                                                                                                                                                                                        │
   │      setExpandedDirs(new Set());                                                                                                                                                                                                         │
   │    }, []);                                                                                                                                                                                                                               │
   │                                                                                                                                                                                                                                          │
   │    const navigate = useCallback((newPath: string) => {                                                                                                                                                                                   │
   │      setCurrentPath(newPath);                                                                                                                                                                                                            │
   │      loadDirectory(newPath);                                                                                                                                                                                                             │
   │    }, [loadDirectory]);                                                                                                                                                                                                                  │
   │                                                                                                                                                                                                                                          │
   │    const toggleSelect = useCallback((filePath: string) => {                                                                                                                                                                              │
   │      setSelectedFiles(prev => {                                                                                                                                                                                                          │
   │        const next = new Set(prev);                                                                                                                                                                                                       │
   │        if (next.has(filePath)) {                                                                                                                                                                                                         │
   │          next.delete(filePath);                                                                                                                                                                                                          │
   │        } else {                                                                                                                                                                                                                          │
   │          next.add(filePath);                                                                                                                                                                                                             │
   │        }                                                                                                                                                                                                                                 │
   │        return next;                                                                                                                                                                                                                      │
   │      });                                                                                                                                                                                                                                 │
   │    }, []);                                                                                                                                                                                                                               │
   │                                                                                                                                                                                                                                          │
   │    const navigateUp = useCallback(() => {                                                                                                                                                                                                │
   │      setCurrentIndex(prev => Math.max(0, prev - 1));                                                                                                                                                                                     │
   │    }, []);                                                                                                                                                                                                                               │
   │                                                                                                                                                                                                                                          │
   │    const navigateDown = useCallback(() => {                                                                                                                                                                                              │
   │      setCurrentIndex(prev => Math.min(files.length - 1, prev + 1));                                                                                                                                                                      │
   │    }, [files.length]);                                                                                                                                                                                                                   │
   │                                                                                                                                                                                                                                          │
   │    const expandDirectory = useCallback((dirPath: string) => {                                                                                                                                                                            │
   │      setExpandedDirs(prev => {                                                                                                                                                                                                           │
   │        const next = new Set(prev);                                                                                                                                                                                                       │
   │        next.add(dirPath);                                                                                                                                                                                                                │
   │        return next;                                                                                                                                                                                                                      │
   │      });                                                                                                                                                                                                                                 │
   │    }, []);                                                                                                                                                                                                                               │
   │                                                                                                                                                                                                                                          │
   │    const collapseDirectory = useCallback((dirPath: string) => {                                                                                                                                                                          │
   │      setExpandedDirs(prev => {                                                                                                                                                                                                           │
   │        const next = new Set(prev);                                                                                                                                                                                                       │
   │        next.delete(dirPath);                                                                                                                                                                                                             │
   │        return next;                                                                                                                                                                                                                      │
   │      });                                                                                                                                                                                                                                 │
   │    }, []);                                                                                                                                                                                                                               │
   │                                                                                                                                                                                                                                          │
   │    const goToParent = useCallback(() => {                                                                                                                                                                                                │
   │      const parent = path.dirname(currentPath);                                                                                                                                                                                           │
   │      if (parent !== currentPath) {                                                                                                                                                                                                       │
   │        navigate(parent);                                                                                                                                                                                                                 │
   │      }                                                                                                                                                                                                                                   │
   │    }, [currentPath, navigate]);                                                                                                                                                                                                          │
   │                                                                                                                                                                                                                                          │
   │    return {                                                                                                                                                                                                                              │
   │      state: {                                                                                                                                                                                                                            │
   │        isOpen,                                                                                                                                                                                                                           │
   │        currentPath,                                                                                                                                                                                                                      │
   │        files,                                                                                                                                                                                                                            │
   │        selectedFiles,                                                                                                                                                                                                                    │
   │        expandedDirs,                                                                                                                                                                                                                     │
   │        currentIndex,                                                                                                                                                                                                                     │
   │      },                                                                                                                                                                                                                                  │
   │      open,                                                                                                                                                                                                                               │
   │      close,                                                                                                                                                                                                                              │
   │      navigate,                                                                                                                                                                                                                           │
   │      toggleSelect,                                                                                                                                                                                                                       │
   │      navigateUp,                                                                                                                                                                                                                         │
   │      navigateDown,                                                                                                                                                                                                                       │
   │      expandDirectory,                                                                                                                                                                                                                    │
   │      collapseDirectory,                                                                                                                                                                                                                  │
   │      goToParent,                                                                                                                                                                                                                         │
   │    };                                                                                                                                                                                                                                    │
   │  }                                                                                                                                                                                                                                       │

   Risk Mitigation



   Potential Issues


   1. Performance with large directories
     • Risk: Slow rendering for 1000+ files
     • Mitigation: Implement virtual scrolling (react-window)
     • Mitigation: Limit initial depth, lazy-load subdirectories


   2. Keyboard input conflicts
     • Risk: File picker shortcuts conflict with Ink's useInput
     • Mitigation: Use conditional input handling (check  isOpen  flag)
     • Mitigation: Document all shortcuts clearly


   3. Cross-platform path handling
     • Risk: Windows backslashes vs Unix forward slashes
     • Mitigation: Use Node's  path  module for all path operations
     • Mitigation: Test on Windows, macOS, Linux


   4. File system permissions
     • Risk: Cannot read certain directories
     • Mitigation: Graceful error handling with user-friendly messages
     • Mitigation: Skip inaccessible directories, show warning
